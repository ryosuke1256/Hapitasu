## 概要

-   Todo アプリを React と Laravel で作ってみた

## 機能

-   タスク一覧表示機能
-   タスク作成機能
-   タスク編集機能
-   タスクチェック機能
-   タスク削除機能
-   サインアップ、ログイン機能
-   タグ、色分け機能

## フロントエンド

-   TypeScript
-   React

※ styled components, Laravel Mix（同一オリジン）

## バックエンド

-   Laravel

## DB

-   MySQL

## インフラ

-   AWS ( VPC, EC2 , RDS, ELB, Route53, CloudWatch)

**サーバー**

ミドルウェア : Nginx

OS 　 : Amazon Linux
